Installation
============

Clone the repo:
`git clone https://github.com/dmitryz/my_vim.git ~/.vim`

Grab Vundles:
`git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

Install Plugins
`vim +PluginInstall +qall`

Copy
`cp ~/.vim/my_vimrc ~/`

Create
`~/.vimrc`
`runtime my_vimrc`
